# Not-Balatro 🎰
Versión simplificada del videojuego Balatro construida en React + Vite
para el curso Sistemas y Tecnologías Web (UVG, 2026).

## Stack
- React 18 + Vite 5
- JavaScript (sin TypeScript)
- CSS puro (sin frameworks externos)

## Cómo correr
```bash
npm install
npm run dev
```
